Come attivarlo

  1. Installa dipendenze: pip install -e ".[summarize]"
  2. Scarica il modello (~2 GB): 
    python scripts/download_summary_llm.py
  3. Verifica che esista models/{{ local_llm_folder }}/{{ local_llm_gguf_file }}
  4. Torna alla home, abilita «Genera riassunto» e scegli LLM locale (Qwen)

Su CPU il riassunto locale è lento (anche 5–15 minuti su audio lunghi). Nessuna API key richiesta.

4. Usare il riassunto

  1. Configura almeno un motore sopra (cloud o locale).
  2. Vai alla home.
  3. Spunta Genera riassunto, scegli il motore e la lunghezza.
  4. Carica i file e accoda.
